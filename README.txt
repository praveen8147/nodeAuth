Node Auth App with PassportJS and JWT
--------------------------------------

Author : Praveen Kumar G


here mainly 5 routes are there,

1. / -> home page

2. /signup -> post API to register user
			  payload = email and password in string JSON object

3. /login -> post API to login user with JWT
			  payload = email and password in string JSON object

4. /logout -> get API for logout and redirected to home page response
